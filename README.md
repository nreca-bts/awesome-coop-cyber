# Awesome Co-op Cybersecurity & Threat Intelligence
[![Awesome](https://awesome.re/badge.svg)](https://github.com/sindresorhus/awesome)

A curated list of **cybersecurity, ICS/OT security, and threat intelligence resources** 
specifically tailored to cooperative electric utilities 
— including training, tooling, and community resources.

😎 Look for the `CoopCyberTech{YY}` in the tag column for resources highlighted during 
the NRECA annual Co-op Cyber Tech conference.

Want to contribute? 💚 Love it! Click the "Contributing" tab or navigate to the CONTRIBUTING.md file to learn how.

## TTP Ecosystem

| Name                                                                                                            | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              | Tags                              |
| --------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------- |
| [Core MITRE ATT&CK(R)](https://attack.mitre.org/)                                                               | Industry-standard framework for adversary tactics, techniques, and procedures. Includes tooling such as the "Navigator" for exploring the framework, including specific threat actors and campaigns. Explore the [Get Started](https://attack.mitre.org/resources/) guide; the [Enterprise](https://attack.mitre.org/matrices/enterprise/), [Mobile](https://attack.mitre.org/matrices/mobile/), and [ICS](https://attack.mitre.org/matrices/ics/) matrices; and their [Data & Tools](https://attack.mitre.org/resources/attack-data-and-tools/) list.                                                                                                                                                                                   | `Framework` `Threat-Intelligence` |
| [MITRE CTID](https://github.com/center-for-threat-informed-defense)                                             | Projects applying ATT&CK to real-world defense (AI, cloud, fraud).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |                                   |
| [CISA Tools](https://github.com/cisagov)                                                                        | CISA has built several tools within or utilizing the TTP Ecosystem, including [Malcom](https://github.com/cisagov/Malcolm) (A network traffic anlysis tool suite, easily deployable for network security monitoring.); [Decider](https://github.com/cisagov/Decider/) (A web application that assists network defenders, analysts, and researchers in the process of mapping adversary behaviors to the MITRE ATT&CK® Framework.); and [Untitled Goose Tool](https://github.com/cisagov/untitledgoosetool) (A robust and flexible hunt and incident response tool that adds novel authentication and data gathering methods in order to run a full investigation against a customer’s Microsoft Entra ID, Azure, and M365 environments.) | `Government`                      |
| [PushSecurity Browser&Identity Attacks Matrix](https://github.com/pushsecurity/browser-identity-attacks-matrix) | A collection of browser and identity ATT&CK techniques covering SaaS apps, browser-based attacks, identity providers, and phishing. It is intended to be a resource for security researchers, red/blue teams, and penetration testers to learn about and share these attack techniques.                                                                                                                                                                                                                                                                                                                                                                                                                                                  | `Identity` `Web Security`         |
| [PushSecurity Phishing Detection Evasion Techniques](https://github.com/pushsecurity/phishing-techniques)       | This project documents a collection of modern phishing detection evasion techniques, breaking down the methods that attackers use at different stages of a phishing attack culminating in account takeover (i.e. stealing sessions, credentials, etc.). Each stage groups the techniques observed against a phase of activity designed to overcome a layer of security control.                                                                                                                                                                                                                                                                                                                                                          | `Phishing`  `Social Engineering`  |
